## 核心功能测试

### 模块生成
- [ ] 点击左侧模块列表，中间输入区显示对应字段
- [ ] 填入有效输入，点击"开始生成"，右侧显示生成结果
- [ ] 每个模块切换后能正常生成结构化内容

### 自定义模块
- [ ] 设置 → 自定义模块 → 新建，填写名称/字段/提示词，保存后模块列表出现
- [ ] 选中自定义模块，填入参数，生成成功

### 对比模式
- [ ] 勾选"对比模式"，选择两个模块，填入输入，点击"双模块生成"
- [ ] 右侧上下分屏显示两个结果

### 批量模式
- [ ] 勾选"批量模式"，输入多行文本，生成后展示多个结果卡片

### 思维导图
- [ ] 生成结果后切换到"导图"视图，显示树形结构图

### Markdown渲染
- [ ] 生成结果后切换到"MD"视图，显示排版后的格式
